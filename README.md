# 👥 User Search App

A simple and responsive React application that allows users to search and filter a list of users in real-time.

---

##  Features

*  Real-time user search
*  Instant filtering using JavaScript
*  Case-insensitive search
*  Displays "No user found" when no match
*  Clean and minimal UI

---

##  Tech Stack

* React (Vite)
* JavaScript (ES6)
* CSS3

---





##  How It Works

* The app stores user data in state using `useState`
* On every input change, it filters users using:

  * `filter()` method
  * `includes()` for matching names
* Updates UI dynamically based on search input

---

##  Deployment

Deployed using Netlify.
